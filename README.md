# 바벨의 도서관 : 마법소녀 레코드

현대 일본 배경 **마법소녀** 세계관의 인터랙티브 AI 소설 — 단일 HTML, 클라이언트 단독.
브라우저가 사용자의 LLM 키로 직접 호출합니다(서버 없음).

**플레이:** https://bizsinsightclub.github.io/newtry_game/

- 우상단 **⚙ 인격 파편 조율**에서 프로바이더(Anthropic·OpenAI 호환·Google Gemini)와 본인 API 키를 입력
- 키는 이 브라우저(localStorage)에만 저장되고 선택한 엔드포인트로만 전송됩니다
- 키 없이 **환영(mock)** 으로 흐름 체험 가능
- 시작 화면에서 주인공 1 + 동행 최대 3을 고르고 이야기를 진행

> `index.html`은 빌드 산출물(데이터·이미지 base64·JS·CSS 인라인)입니다.
> 원본 소스/빌더는 별도 저장소에서 관리됩니다.
